## Strategy context
GF-geo route: the geometric `thm:generic_flatness` (`AlgebraicGeometry.genericFlatness`) wraps the
now-DONE algebraic core `thm:generic_flatness_algebraic`. The geometric assembly (Nitsure §4 form):
restrict to a non-empty affine open `Spec A ⊆ S` (A a noetherian domain, since S is integral +
locally noetherian); cover `p⁻¹(Spec A)` by finitely many affine `W_j = Spec B_j` (each finite-type
over A); read `M_j = Γ(F, W_j)` finite over `B_j`; apply the algebraic core per patch to get `f_j ≠ 0`
with `(M_j)_{f_j}` free over `A_{f_j}`; take the common basic open `V = D(∏ f_j)`; conclude flatness
from freeness via the flat-at-every-maximal criterion. Two steps rely on genuine missing-Mathlib bridges
(G1, G3) that the iter-023 prover terminated honestly against. This iter we blueprint them as proper
lemma blocks so a `mathlib-build` prover can target G1 (the witness-blocking gap).

### FIX 3 — add lemma block G1 (the keystone bridge; blocks the witness)
New lemma `lem:gf_qcoh_fintype_finite_sections` with
`\lean{AlgebraicGeometry.gf_qcoh_fintype_finite_sections}` (or a name in the `AlgebraicGeometry`
namespace you find cleaner — note it for the plan agent). Statement, in project notation:

> Let `X` be a scheme and `F : X.Modules` with `[F.IsQuasicoherent]` and `[F.IsFiniteType]`. Then for
> every affine open `W ⊆ X`, the section module `Γ(F, W)` is a finite module over `Γ(X, W)`.
> (Lean shape: `∀ {W : X.Opens}, IsAffineOpen W → Module.Finite Γ(X, W) Γ(F, W)`.)

Informal proof (the genuine content): over the affine open `W ≅ Spec Γ(X,W)`, quasi-coherence of `F`
identifies `F|_W` with the associated sheaf `Ñ` of the section module `N = Γ(F, W)` (the affine-local
identification `F|_W ≅ Ñ`; sections over basic opens `D(g) ⊆ W` are the localizations `N_g`, cf. the
QUOT-chapter node `lem:qcoh_section_localization_basicOpen`,
`AlgebraicGeometry.Scheme.Modules.isLocalizedModule_basicOpen`). The `IsFiniteType` hypothesis says `F`
is locally generated by finitely many sections; on the affine `W` (quasi-compact) this gives a finite
generating set of `N = Γ(F, W)` over `Γ(X, W)`, i.e. `N` is a finite `Γ(X,W)`-module.

finiteness.` Add a `% NOTE: overlaps the QUOT keystone lem:qcoh_section_localization_basicOpen — to be

state the qcoh-affine-local identification as a sub-step.
